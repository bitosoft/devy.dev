## Installation

Ensure you have Node.js (version 12 or higher) installed on your system. Then, you can install Devy globally using npm:

```bash
$ npm install -g @bitosoft/devy
```

## Quick Start

After installation, you can start using Devy by running:

```bash
$ devy hello world
```
